Core Features

Everything you need,
nothing you don't

Node Concatenation
Connect clips with a drag. Each node represents a segment — link them in any order to define your final video sequence visually.
Batch Processing
Run the same node graph across dozens of source files at once. One template, infinite variations — ideal for social content creators.
Template Mode
Build a reusable node template with named slots (hook, intro, outro). Each batch set fills the placeholders automatically.
Cron Jobs
Schedule renders to run at specific times. Queue up overnight processing and wake up to finished videos in your export folder.
Split Screen Node
A dedicated split-screen node lets you composite two streams side by side — perfect for reaction videos, comparisons, and tutorials.
Smart Duplication
Duplicate any node chain with a single click. Reuse segments across sets without re-importing or re-configuring anything.
